# 鎖在外面、留在那裡

私有化、政治化和柱狀化，是迷惑信仰的三角陷阱。

{{% hint info %}}
現代信仰面臨一個致命的三角陷阱：私有化、政治化與柱狀化。
呼召要求我們打破這些圍牆，不僅在私底下虔誠，更要在公共領域中展現信仰的整體性。若是做鹽，就不能只留在鹽罐裡；若是做光，就不能被關在黑暗之外。
{{% /hint %}}

---

## 一、迷惑信仰的三角陷阱

作者指出，當代基督徒在試圖影響社會時，常落入三種結構性的陷阱。這三者互相關聯，削弱了信仰的力量：

| 陷阱類型                          | 定義                                                                                   | 後果                                                                               |
| :-------------------------------- | :------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------- |
| **1. 私有化**<br>(Privatization)  | 現代化在公眾和私人生活間製造裂縫，將信仰限制在「私人領域」（個人的自由、自我實現）。   | **整體性喪失**。主的權力被縮小到如同「口袋」一般，全部的標準變成了部分的價值。     |
| **2. 政治化**<br>(Politicization) | 這是對私有化的直接反動。教會試圖奪回發言權，卻單向認同政治運動的左翼或右翼。           | **失去抗衡**。信仰淪為政治的附庸，這是現代歐洲拒絕基督教會的主要原因之一。         |
| **3. 柱狀化**<br>(Pillarization)  | 鼓勵每個信仰團體在自己領域中，建立自己的機構及網狀組織（如：基督徒黃頁、基督徒學校）。 | **失去轉化力**。信徒被隔離在自己的同溫層（柱子）裡，削弱了對主流社會的滲透與轉化。 |

---

## 二、呼召的回應：突圍之道

針對上述三個陷阱，呼召提供了具體的解藥：

1.  **抵抗私有化 $\rightarrow$ 堅持整體性**

    - 拒絕將信仰切割為「私人的」與「公眾的」。
    - 在工作場所和家中，信仰必須保持一致。

2.  **抵抗政治化 $\rightarrow$ 建立抗衡機制**

    - 要求每個屬於人的忠誠和關係，都須有抗衡（Tension）的機制。
    - 不完全照單全收任何政黨的意識形態，保持先知性的批判距離。

3.  **抵抗柱狀化 $\rightarrow$ 投入與轉化**
    - 拒絕自我隔離在基督徒的次文化中。
    - 要求我們投入社會主流，發揮轉化力量。

## 三、該柏爾 (Abraham Kuyper) 的榜樣

_註：書中提到的該柏爾（或譯凱波爾），是荷蘭首相兼神學家，他的一生完美詮釋了如何不被「鎖在教會裡面」。_

- **鎖在外面**：他的一生彷彿被「鎖在教會外面」，但他沒有因此退縮。
- **留在那裏**：他選擇留在公共領域（政治、教育、媒體），在那裡活出呼召，證明了基督的主權及於萬有。

---

## 四、小結：鹽與光的檢視

- 你的信仰是否在私底下很投入，卻與社會無關？
- 你所參與的團體和你對它的忠誠，與你對基督的委身是否成正比？
- **關鍵拷問**：你是做光做鹽，還是必須被關在基督徒的圈子外？

## 五、研讀與默想 (Action Plan)

### 📌 研讀指引

1.  **歷史榜樣**：該柏爾（Abraham Kuyper）如何活出「鎖在教會外面」的一生？這對不想當牧師、想在職場事奉的你有何啟發？
2.  **私有化的騙局**：今日社會中有哪些跡象顯示信仰被「私有化」了？（例如：信仰只是心靈寄託，不能談論公共議題）。這與呼召有何正面衝突？
3.  **政治的試探**：你有看到哪些信仰「政治化」的例證？當教會過度依附特定政黨時，對社會的影響是什麼？我們如何採取政治行動（關心公義），卻不會將教會政治化（淪為打手）？
4.  **同溫層危機**：你發現社會的哪個部分特別容易受「柱狀化」的誘惑？（例如：只讀基督徒寫的書、只去基督徒開的店）。這對「做鹽做光」構成了什麼挑戰？

### 📖 聖經默想：馬太福音 5:13-16 (鹽與光)

- **本質**：鹽和光的本質是甚麼？（鹽要防腐調味，光要照亮黑暗）。
- **環境**：甚麼環境下這些本質能更活躍？（鹽必須接觸腐肉，光必須進入黑暗）。
- **應用**：這對我們該如何為人（不應隔離自己）有何意義？
