                                                            揮之不去的問題 #

                                                            我是誰？我為誰而活？只有當我們回應基督的呼召，我們才能成為真正的自我。

                                                            「我是誰？」這個揮之不去的問題，無法單靠社會框架（受限制）、自我意志（勇敢）或宿命論（被建造）來回答。潘霍華在獄中給出了終極答案：「無論我是誰，祢知道，主，我屬於祢。」

                                                            一、獄中的省思：責任與自我 #

                                                            當自由被剝奪，關於存在的思考反而更為清晰。作者列舉了西方三大獄中書信經典：

                                                            1. 哈維爾 (Václav Havel)：《獄中書簡》(Letters to Olga)
                                                            2. 潘霍華 (Dietrich Bonhoeffer)：《獄中書簡》(Letters and Papers from Prison)
                                                            3. 波伊丟斯 (Boethius)：《哲學的慰藉》(Consolation of Philosophy)

                                                            哈維爾的信念：人類的奧秘關乎責任 #

                                                            哈維爾認為，自我責任是塑造身分的關鍵工具：

                                                            • 像把刀：在存有全貌中雕刻出獨特特色。
                                                            • 像隻筆：在存有歷史中寫下全新的故事。

                                                            二、現代身分認同的三種危機 #

                                                            現代人試圖回答「我是誰」時，通常落入以下三種態度，但都有其缺陷：

                                                            1. 受限制的生命 (The Constrained Life) #

                                                            • 定義：以普遍方式解釋人類的個別性，強求一致。
                                                            • 方法：修剪人格，以配合大量生產的框架（Procrustean bed）。
                                                            • 後果：人被迫躺在強求一致的床上，失去獨特性。

                                                            2. 勇敢的生命 (The Courageous Life) #

                                                            • 定義：人皆有自由成為想成為的，僅需要勇氣和意志力（如尼采、濟慈）。
                                                            • 主張：「人就是自己的作者」。
                                                            • 荒謬處：
                                                              • 即使能做想做的事，問題仍在：我們到底想做什麼？
                                                              • 即使用盡意志，我們可能仍無法成為想要的樣子。

                                                            3. 被建造的生命 (The Constructed Life) #

                                                            • 定義：生命已被定型，擁有既定的劇本（受新世紀思潮影響）。
                                                            • 主張：命運、預定、靈魂伴侶。
                                                            • 問題：若一切皆已寫好，人的自由意志何在？

                                                            三、呼召：超越與整合 #

                                                            「蒙召的生命」並非完全否定上述觀點，而是給予了更高的詮釋：

                                                            1. 我們確實受限，但不是「受限制」，而是**「受蒙召」**（了解塑造我們的力量是寶貴的）。
                                                            2. 我們必須勇敢，但回應呼召遠比憑空撰寫自我劇本更有意義。
                                                            3. 我們是被建造的，但蒙召更強調自由及未來，而非宿命的綑綁。

                                                            四、潘霍華的詩：我是誰？ #

                                                            這首詩深刻描繪了外在形象與內在真實的衝突，以及最終的歸屬。

                                                            《我是誰？》

                                                            我是誰？別人經常告訴我 我會走出監獄的束縛 平靜地、愉悅地、堅定地，就像鄉紳走出自己的宅邸。

                                                            我是誰？別人經常告訴我 我可以和我的獄卒交談 自由地、友善地、清楚地，好像指揮掌權的人是我。

                                                            我是誰？別人也告訴我 我將承受不幸的年月 平靜地、微笑地、驕傲地，就像一個習於得勝的人。

                                                            那麼，我真的是別人所說的那個人嗎？ 或著我只是我所知道的那個自己，不安、飢渴、又生病，像籠中鳥一樣， 為呼吸而掙扎，好像有手扼住我的喉嚨，渴望色彩，渴望花朵，渴望鳥群的聲音， 渴求仁慈的話語，渴求鄰舍，因暴政和微小的羞辱而憤怒顫抖， 困在對偉大事物的期盼中，無力地為遠方的朋友悲傷， 在祈禱、思考、有作為時，感到疲倦、空虛、衰弱，準備向一切告別？

                                                            我是誰？是這人還是那人？我是否今天是這人，明天是那人？ 我是否同時是這兩種人？在他人面前偽善，在自己面前則是可鄙而悲哀的弱者？ 或是在我裡面仍有東西像被打敗的軍隊，在已經得到的勝利面前倉皇逃跑？

                                                            我是誰？這些寂寞問題嘲笑著我。 無論我是誰，祢知道，主，我屬於祢。

                                                            五、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 反思責任：哈維爾認為「責任」與「自我認同」有何關聯？這對你有何啟發？
                                                            2. 檢視風格：為何許多強調「自我風格」的答案，無法滿足內心最深的渴望？
                                                            3. 關於勇敢：對於「勇敢生命」的觀點，你如何決定你真正想要甚麼？
                                                            4. 關於宿命：「被建造的生命」觀點有哪些限制？
                                                            5. 回應呼召：「蒙召的生命」如何超越上述三種觀點？若你以神的眼光看自己，生命將會有何改變？
