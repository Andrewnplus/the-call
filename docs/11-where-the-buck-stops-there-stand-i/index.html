                                                            堅立在責任之處 #

                                                            對現代、世俗、愛好自由的人而言，對神負責根本就不被列入考慮。

                                                            齊克果曾說：「人類所有的精明都為尋求『能不負責任地活著』。」 但呼召提醒我們：我們必須自發地向唯一的觀眾（上帝）負責。這不是沈重的律法，而是身為人的尊嚴——我們是被創造來「許下承諾」的動物。

                                                            一、現代責任感的危機：日落價值觀 #

                                                            現代人愛好自由，卻極少考慮「對神負責」。作者用了一個深刻的比喻來形容現代的責任感：

                                                            • 日落價值觀：責任感就像日落時的色彩，雖然最為繽紛，卻是因為它正在消逝。
                                                            • 斷根的危機：現代責任感已與其根源（上帝）剝離。結果是，責任不再是回應呼召的喜悅，反而成了沈重的道德重擔。

                                                            畢卡索的悲劇：女神與腳踏墊 #

                                                            世所公認的藝術大師畢卡索，在人際關係上卻留下了著名的分類法，反映了缺乏永恆座標的愛是多麼脆弱：

                                                            1. 女神 (Goddesses)：起初的崇拜對象。
                                                            2. 腳踏墊 (Doormats)：遲早，第一種人都會淪落為這一種。

                                                              啟示：若沒有永恆作為參考點，人類的承諾與熱情終將枯竭。

                                                            二、最後一句話才最重要 #

                                                            這是一場關於真理的戰爭：

                                                            • 世俗的謊言：希特勒的宣傳部長（Goebbels）曾說：「策略皆如『率先說出第一句話者永遠是對的』。」
                                                            • 信仰的真理：他錯了。最後一句話才最重要。
                                                            • 不可逃避的選擇：我們可以為是否有自由選擇權爭辯，但我們沒有自由「不」選擇。不選擇本身，就是一種選擇。

                                                            三、齊克果與尼采的洞見 #

                                                            兩位哲學家從不同角度探討了責任的本質：

                                                            • 齊克果 (Søren Kierkegaard)：
                                                              • 批判：「人類所有的精明都為尋求『能不負責任地活著』。」
                                                              • 定義：「讓人永為生活每刻負責，即便是最後一件事——此為基督教信仰。」
                                                            • 尼采 (Friedrich Nietzsche)：
                                                              • 在《道德系譜學》中指出：人類責任的追尋是「繁衍能作承諾的動物的工作」。能夠許下承諾並信守，是人之所以為人的特質。

                                                            四、結語：回應神的問話 #

                                                            真正的負責，不是對規條負責，而是對「人」負責——對那位上帝負責。 你是否願意堅立，成為一個負責的人？ 試圖用自己的一生，來回應神的問話及呼召。

                                                            五、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 愛的保存期限：為什麼像畢卡索這樣的大師，仍需要在教堂祈求不朽的愛情？為何所有的應許若不以「永恆」為參考點，最終都會變質？
                                                            2. 責任的重擔：為何現代社會強調的責任感（環保、公民責任等），有時讓人感到疲憊卻無力矯正道德危機？（提示：因為缺乏了對神的敬畏作為根基）。
                                                            3. 對象的區別：「為某事負責」（Accountable for something）和「向某人負責」（Accountable to someone）有何本質上的不同？
                                                            4. 獨處時的誠信：當身旁沒人時，意識到「向唯一觀眾（上帝）負責」如何幫助你避免陷入不負責任的試探？
