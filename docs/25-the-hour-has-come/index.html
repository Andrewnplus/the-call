                                                            時候已經來臨 #

                                                            合神心意的人準備好要覺察時代的徵兆，並在自己的世代中照神的旨意事奉祂。

                                                            英國政治家狄斯雷利（Benjamin Disraeli）曾說：「能認識時代的精神及認識自己，是成功的秘訣。」 合神心意的人必須準備好察覺「時代的徵兆」（Signs of the times），並擁有選擇時機的智慧，在自己的世代中照神旨意侍奉祂。

                                                            一、歷史的智慧：掌握時機 #

                                                            歷史上的偉大領袖都深知時機的重要性：

                                                            • 凱撒（Julius Caesar）：他在盧比孔河（Rubicon）畔的決斷，顯示了選擇時機是成就霸業的關鍵。
                                                            • 政治家的直覺：無論是狄斯雷利提到的「時代精神」，還是俾斯麥形容的「抓住神的外袍」，都指向同一種能力——在關鍵時刻採取行動。

                                                            二、釐清耶穌身分：檢驗的原則 #

                                                            要了解耶穌如何掌握呼召的時機，首先必須釐清我們對祂的認識。作者提出了四個檢驗原則：

                                                            1. 無特權：基督徒不必要求特別的辯護；信仰必須和其他宗教一樣接受檢驗。
                                                            2. 歷史證據：查驗的最佳方式是檢視歷史證據。（錯誤觀點的問題不在於冒犯，而在於它是虛構的）。
                                                            3. 雙向途徑：
                                                              • 向前：從猶太文明的期待來查驗。
                                                              • 向後：從福音的證據來查驗。
                                                            4. 三個核心問題：任何關於「耶穌是誰」的結論，都必須回答：
                                                              • 耶穌如何看待祂自己？（最關鍵）
                                                              • 耶穌為何而死？
                                                              • 如何解釋基督教會的興起？

                                                            三、耶穌的自我認知與時機 #

                                                            耶穌清楚視自己的生命目標來自神的呼召，這決定了祂對時機的掌握：

                                                            • 先知：宣告以色列的流亡已結束，神的國度已降臨。
                                                            • 彌賽亞：一位人形化的「新以色列」。
                                                            • 雅威（Yahweh）：祂的言行彷彿真的是以色列的神。

                                                            四、呼召中的時機智慧：四項主題 #

                                                            耶穌的教導揭示了呼召與時機的密切關係：

                                                            1. 倚靠神：呼召關乎的是全然倚靠神的時間表。
                                                            2. 拒絕錯誤方法：呼召關乎的是拒絕用不適當的方法（如權謀、暴力）來獲得正確的時機。
                                                            3. 準備就緒：呼召關乎的是將回應神的能力所表現出的順服（隨時待命）。
                                                            4. 決心：呼召關乎的是在關鍵時刻的果斷決心。

                                                            五、小結：時間在祂手中 #

                                                            • 你是否試圖依照自己的步調行事，想憑己力解開生命的謎？
                                                            • 你是否用自己的「先知團」（顧問、數據）來解讀你的預言？
                                                            • 或者，你願意將眼光放在神身上，並且確定你的時間在祂的手中？

                                                            聆聽拿撒勒人耶穌的聲音，回應祂的呼召。

                                                            六、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 盧比孔河時刻：你生命中是否有過「度過盧比孔河」（破釜沈舟、無法回頭）的關鍵時刻？你是否及時完成了該做的事？
                                                            2. 描述時機：狄斯雷利、俾斯麥和邱吉爾用「時代精神」、「抓住神的外袍」等意象來描述關鍵時刻。這些比喻帶給你什麼啟發？
                                                            3. 耶穌的超越性：為什麼檢驗耶穌呼召的來龍去脈如此重要？祂從神領受的呼召為何具有如此激進的超越性？
                                                            4. 四種面貌：作者提出的四項主題（倚靠、拒絕錯誤方法、準備就緒、決心），哪一項對現在的你最具挑戰性？
                                                            5. 順服與時機：你如何理解「時機」？耶穌期待我們在某些時刻選擇順服的行動，這對你目前追尋呼召的步調有何修正？

                                                            📖 聖經默想：馬太福音 25:1-13 (十個童女的比喻) #

                                                            • 智慧的角色：在這比喻中，「選擇時機的智慧」扮演了什麼角色？（油的預備代表了什麼？）
                                                            • 錯失的代價：新郎對愚拙童女的反應（我不認識你們），告訴我們該如何看重時間與機會？
                                                            • 警醒：既然我們不知道人子再來的時刻，我們應當保持怎樣的生活態度？
