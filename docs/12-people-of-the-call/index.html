                                                            蒙召的子民 #

                                                            委身於集體呼召，意指我們必須在一切個別呼召中，尊榮基督教會的目標及利益。

                                                            法國政治家塔列朗（Talleyrand）曾說：「沒有個人，甚麼事都不會發生；沒有組織，就沒有東西能存活。」 蒙召不只是個人的事，我們必須委身於「集體呼召」，在追求個體卓越的同時，尊榮基督身體（教會）的目標。

                                                            一、現代社區的崩解：柴契爾夫人的爭議 #

                                                            現代社會面臨著社區觀念的危機。英國前首相柴契爾夫人曾有一句著名的自我辯護：

                                                            「沒有社會這回事，只有個人和他們的家人。」

                                                            這句話同時激怒了兩派人，也揭示了現代人的困境：

                                                            • 自由主義者不悅：因為她用自由主義的邏輯來攻擊自由主義的社會福利觀。
                                                            • 保守派驚慌：因為這忽視了構成我們生存世界豐富背景的「群體」與「傳統」。

                                                            現代困境的三個特徵：

                                                            1. 觀念削弱：現代社區觀念與傳統相比已大幅削弱。
                                                            2. 機構偏見：人們對所有機構（尤其是大型機構）都帶有根深蒂固的懷疑。
                                                            3. 教會稀釋：因各類「志願組織」（福音機構）興起，信徒對「整體教會」本質的概念反而變得更加薄弱。

                                                            二、回應集體呼召的挑戰 #

                                                            要活出蒙召的子民身分，我們必須克服以下挑戰：

                                                            1. 抵擋個人主義：必須毅然決然地抵擋現代漫不經心的個人主義傾向（例如激增的宗派主義）。
                                                            2. 尊榮教會利益：在一切的「個別呼召」中，不能損害，反而要尊榮基督教會的目標及利益。
                                                            3. 分辨絕對與相對：
                                                              • 將神的絕對變成相對，或是將相對的事物變成絕對，都是錯誤。
                                                              • 柴斯特頓（Chesterton）的警語：「如果有件事比現代主要道德觀的敗落更為惡劣，那就是強調次級道德。」
                                                            4. 持續更新：要牢記我們必須不斷地更新，甚至是為了「要更新而更新」。

                                                            三、反思：教會 vs. 生活風格聚集地 #

                                                            這是一個尖銳的自我檢視：

                                                            • 你是否常對「機構型教會」感到沮喪，幻想著有所謂「非機構型教會」的烏托邦？
                                                            • 你與其他信徒的聚集，是基於神聖的呼召，還是只是一種屬靈的**「生活方式聚集地」（Lifestyle Enclave）**——一群品味相投的人聚在一起取暖？

                                                            真正的忠誠，應以那個非政治性、非種族性，聚集萬國萬民的團體為目標。

                                                            四、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 無名英雄：書中提到二戰期間法國尚朋村（Le Chambon）的居民救助猶太人卻毫不居功。若有群體像他們一樣，為何別人的稱讚反而會讓他們感到不舒服？（提示：因為他們認為這只是做了該做的事）。
                                                            2. 機構的兩難：志工組織和福音機構雖有利於社會，但為何會逐漸侵蝕對「教會本質」的認識？我們該如何在「支持機構」與「委身教會」間取得平衡？
                                                            3. 教會跳槽風：「集體呼召」的觀念，對現代信徒動輒更換教會（Church Hopping）的現象提出了什麼呼籲？
                                                            4. 個人與群體：你是否認為即使脫離了集體呼召（教會），仍能實現個人的呼召？為什麼？

                                                            📖 聖經默想：哥林多前書 12:1-27 (基督的身體) #

                                                            • 肢體與功能：保羅如何用「身子」與「肢體」的關係，來說明不同恩賜的運用？
                                                            • 互相關聯：為何他選擇用「有機的身體」而非「組織圖」來描述信徒間的關係？這對你的教會生活有何啟示？
