                                                            與你何干？ #

                                                            我們看到別人的事奉得到神的祝福，就懷疑自己的價值。嫉妒的最終目標，是指責神不應該給別人祝福。

                                                            當我們看別人的事奉得到祝福，就開始懷疑自己的價值，這就是嫉妒。 嫉妒的最終目標極其可怕——它不僅是對別人的好感到悲傷，更是指責神不應該給別人祝福。對此，耶穌只有一句話：「與你何干？（What is that to thee?）」

                                                            一、嫉妒的本質：悲傷與指控 #

                                                            呼召的真理與「天分」、「渴望」以及幾乎無法逃避的「嫉妒」緊密相連。

                                                            • 定義：神學家阿奎那（Thomas Aquinas）將嫉妒定義為**「對別人的好覺得悲傷」**。
                                                            • 心理機制：我們習慣向四周觀看，查看那些與我們呼召相近的人。當覺得他們的成功超過所配得，而自己的成功卻不足時，失望與憤怒便油然而生。

                                                            二、為何呼召對嫉妒毫無招架之力？ #

                                                            作者指出，嫉妒之所以能輕易摧毀蒙召的人，有三個主要原因：

                                                            1. 打擊核心：嫉妒正中我們的恩賜及最深處的渴望與呼召交會之處。（我們通常只嫉妒那些我們也想擁有的才能）。
                                                            2. 引發爭競：嫉妒是引起爭競的要素，它將原本神聖的呼召腐化為世俗的比較。
                                                            3. 褻瀆上帝：嫉妒特別攻擊呼召，因為呼召是直接回到神的面前。嫉妒別人，本質上是在褻瀆神，質疑分配恩賜的那位主宰犯了錯誤。

                                                            三、案例：薩里耶利對決莫札特 #

                                                            書中引用彼得·謝弗（Peter Shaffer）的劇本《阿瑪迪斯》（Amadeus），描繪了嫉妒最駭人的樣貌：

                                                            • 薩里耶利（Salieri）的痛苦：他看見莫札特（Mozart）這個粗俗的人竟然擁有上帝神聖的恩賜，他感到被神背叛。
                                                            • 對神的痛罵：他的嫉妒不只是針對莫札特，更是針對上帝。他無法理解為何神將才華賜給這樣的人，卻不賜給渴慕事奉神的自己。
                                                            • 失敗的復仇：這種被扭曲的景象，最終導致了絕望與毀滅。

                                                            四、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 恩賜的重擔：為何恩賜有時會成為重擔？我們怎樣可以減輕這樣的痛苦？
                                                            2. 薩里耶利的悲劇：在莫札特與薩里耶利的故事中，薩里耶利為何痛罵上帝？這反映了嫉妒背後隱藏著什麼樣的神學危機？
                                                            3. 社會觀察：在社會的哪些範疇裡（如職場、政治），你看到被「失敗的復仇」所扭曲的景像？為何嫉妒特別能敗壞民主制度？
                                                            4. 罪的嚴重性：你是否同意嫉妒是僅次於驕傲的第二嚴重的罪？
                                                            5. 解藥：在你生活的哪個層面你曾嫉妒他人？那個經驗如何影響你跟上帝的關係？你認為嫉妒的解藥是什麼？（提示：回到「與你何干」的專注，單單跟隨主）。
