                                                            道的追隨者 #

                                                            基督徒在世上旅行，像一個穿越時間的朝聖者，尋找永恆的王國。

                                                            許多人被「基督」吸引，卻對「基督教」感到無聊或厭惡。 將焦點從「基督」轉移至「基督教」，本身就是腐敗的記號。唯有將生命視為一場旅程，成為「道的追隨者」，才能抵抗這種沉淪。

                                                            一、信仰的沉淪：從基督到基督教 #

                                                            作者觀察到一個令人憂心的歷史規律，即信仰焦點的偏移與僵化：

                                                            1. 現象：人們對耶穌感興趣，但對教會體制（基督教）反感。
                                                            2. 腐敗的軌跡：
                                                              • 基督 (Christ)：活潑的關係與源頭。
                                                              • $\downarrow$
                                                              • 基督徒 (Christian)：跟隨者。
                                                              • $\downarrow$
                                                              • 基督教 (Christianity)：僵化的組織與教條。
                                                            3. 關鍵指標：信仰的聲明與實踐之間是否一致？若言行不一，信仰就只剩下空殼。

                                                            二、抵抗腐敗：重尋「道」 #

                                                            如何防止上述的沉淪？答案在於「呼召」的動態本質。

                                                            • 真正的跟隨：只有當我們正在跟隨基督（現在進行式），才算是跟隨基督的人。
                                                            • 旅程的視角：
                                                              • 基督徒在世上旅行，像個穿越時間的朝聖者，尋找永恆的王國。
                                                              • 視生命為一場旅程：旅程未到最後，不能做最後的定論。
                                                              • 寬容：認知每個人處在不同的旅程階段，耶穌的跟隨者其實比我們想像的還多。

                                                            三、離鄉者 vs. 流浪者 #

                                                            在尋找家鄉的旅程中，我們必須釐清自己的身分：

                                                            • 流浪者 (Wanderer)：沒有目的地，隨波逐流。
                                                            • 離鄉者/朝聖者 (Exile/Pilgrim)：知道自己有家鄉，只是暫時客居在此。
                                                            • 思鄉病：人類生命深處都在尋找家鄉，這種「神聖的思鄉病」是推動我們前進的動力。

                                                            四、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 區分本質：「團結的信仰」（有機體）和「組織形態的信仰」（機構）有何不同？為何我們常混淆兩者？
                                                            2. 防止墮落：為何信仰容易變成平淡無奇的例行公事？你認為防止信仰從「跟隨基督」墮落成「基督教」的關鍵是什麼？
                                                            3. 面對批評：世人常攻擊基督教，這僅是粗暴的偏見，還是綜觀歷史後的一個恰當評估？我們該如何回應？
                                                            4. 名詞的重量：試著用「耶穌的跟隨者」來代替「基督徒」這個稱呼，這對你而言只是語意上的變化，還是有助於重新定義你的身分？
                                                            5. 思鄉的動力：「思鄉病」對屬靈生命意味著什麼？為何忘記「生命是趟未完成的旅程」會有危險？

                                                            📖 聖經默想：約翰福音 14:1-6 #

                                                            耶穌說：「我就是道路、真理、生命…」

                                                            • 安慰：耶穌安慰門徒的話（我去原是為你們預備地方），與我們的生命之旅有何關聯？
                                                            • 終點：試著描繪旅程終點——天堂（父家）的景象。這如何影響你今天的腳步？
