                                                            對抗正午的惡魔 #

                                                            現代人的悲劇不是因為他對自己生命的意義所知越來越少，而是生命的意義越來越不成為他的困擾。

                                                            現代人的悲劇，不是因為他對自己生命的意義所知越來越少，而是生命的意義越來越不成為他的困擾。這種對意義的冷漠，就是「正午的惡魔」（懶惰）。

                                                            一、懶惰（Accidie）的三個切入點 #

                                                            這裡的「懶惰」並非指整天睡覺不做事，而是一種屬靈的麻木與厭倦。呼召與這種狀態完全對立。作者指出懶惰透過三個層面侵蝕現代人：

                                                            1. 哲學層面：信心的失落 #

                                                            • 現象：失去對神的信心。
                                                            • 後果：當源頭斷絕，生命本身的活力（Vitality）也隨之被腐蝕。

                                                            2. 文化層面：物力論的假象 #

                                                            • 誤解：現代人以為現代世界的興起代表著充滿活力。
                                                            • 真相：那其實只是物力論（Dynamism）——一種藉由不斷忙碌、活動來掩蓋內心空虛的狀態。外表忙碌，內在卻是癱瘓的。

                                                            3. 生命歷程：中年危機 #

                                                            • 時刻：通常發生在人生的「正午」（中年）。
                                                            • 症狀：渴望成功的事業、滿足的工作、豐富的個人生活，但當這些都達成或無法達成時，內心卻感到巨大的無聊與無力。

                                                            二、梅爾維爾的嘆息 #

                                                            《白鯨記》作者梅爾維爾（Herman Melville）曾寫信給朋友抱怨：

                                                            「金錢對我施了咒詛。」

                                                            這句話道出了許多人的心聲：當工作僅僅是為了維持生計或獲取金錢，而缺乏更高的呼召時，工作就成了詛咒，人也因此失去了對抗「正午惡魔」的力量。

                                                            三、呼召：唯一的解藥 #

                                                            如何從這種心靈的麻痺中甦醒？呼召與職涯考量必須有正確的優先次序：

                                                            1. 優先性：呼召必須在職涯考量之先。
                                                            2. 透視：藉著呼召的透視，我們才能看清自己最深刻的恩賜和抱負。
                                                            3. 滿足：只有在呼召中，我們才能在工作中找到最深刻的滿足，因為我們知道是為誰而做。

                                                            四、小結：渴望甦醒 #

                                                            你是否渴望逃離一個「缺乏比自己更高遠目標」的貧乏生活？

                                                            • 想要超越周圍眾多平凡、平庸而且相當絕望的事物？
                                                            • 想知道一個任何勢力都無法威嚇、任何失敗都不能令你沮喪的目標？

                                                            聆聽耶穌的聲音，回應祂的呼召，這是擊退正午惡魔的唯一武器。

                                                            五、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 定義懶惰：為何「正午的惡魔」是對懶惰（屬靈的厭倦）最恰當的描述？在我們的生命中，哪個時刻最易受到這種誘惑？
                                                            2. 無聊感：你曾否對生活工作感到如此無聊和無力？你過去如何對抗這種情況？
                                                            3. 連鎖反應：對神失去信心，如何導致對任何事都失去信心？你的生活經驗是否印證這種說法？
                                                            4. 文化的暴力：現代的懶惰（內在空虛）如何反而助長了外在文化的暴力和破壞行為？（提示：因為人需要更強烈的刺激來感覺自己活著）。
                                                            5. 職涯規劃：呼召如何幫助我們重新看待職涯？

                                                            閱讀所羅門王的追尋之旅：

                                                            • 嘗試：所羅門王到哪些地方追尋生命的意義？（喜樂、美酒、工程、財富、音樂…）。
