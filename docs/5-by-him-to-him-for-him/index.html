                                                            藉著祂、走向祂、為了祂 #

                                                            工作狂藉著工作來尋找意義，帶著使命感的人將意義賦予工作。

                                                            若缺少深刻的呼召感，工作或事業都不能完全令人滿足。現代工作倫理最大的危機，在於變成了一種「沒有呼召者的呼召」（Calling without a Caller）—— 只剩下工作的責任，卻遺忘了發出呼召的主。

                                                            一、更正教的錯誤：世俗形態的二元論 #

                                                            前一章提到了天主教「聖俗二分」的錯誤，本章則聚焦於更正教（新教）後來發展出的另一種錯誤：高舉世俗、貶抑靈性。

                                                            • 起源：早期的清教徒並未犯此錯，他們從未想將主要呼召與次要呼召分開。
                                                            • 變質：隨著時間推移，信仰與呼召逐漸分離，導致帶有負面色彩的職業觀出現。
                                                            • 危機：「呼召」不再由神的命令決定，轉而由社會責任與角色來導引。

                                                            二、關鍵概念：沒有呼召者的呼召 #

                                                            這是本章最犀利的批判。當我們把「呼召」等同於「職業」或「工作」時，就產生了這種怪異的現象：

                                                            1. 失去源頭：人們努力工作，卻忘了是「誰」在呼召。
                                                            2. 失去滿足：誠如史達・特寇（Studs Terkel）所言：「多數人…都對生活中的工作勉強接受或不喜歡」。

                                                            三、如何從錯誤中轉回？ #

                                                            要修復這種破裂的關係，必須採取以下步驟：

                                                            1. 丟棄錯誤觀念：徹底揚棄「沒有呼召者的呼召」這種看法。
                                                            2. 恢復主要呼召的地位：
                                                              • 敬拜（背景）：恢復呼召的敬拜背景。
                                                              • 委身（核心）：恢復對主的委身。
                                                              • 原則：不獻身給任何人、事物超過主自己。

                                                            四、持守平衡：再次檢視兩種錯誤 #

                                                            為了確保我們走在正路上，需再次釐清這兩種歷史性的偏差：

                                                            • 天主教的錯誤（前一章重點）：不強調次要呼召 $\rightarrow$ 造成避世。
                                                            • 更正教的錯誤（本章重點）：不強調主要呼召 $\rightarrow$ 造成世俗化。

                                                            最終挑戰：你是否願完全、不計代價地永屬於祂，以至於次要事仍為次要，首要事永居首位？

                                                            五、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 工作的不滿：書中提到的諾拉華生（Nora Watson）對工作的混亂，反映了她期望從工作中獲得什麼卻不可得？你何時對工作有過類似「勉強接受」的感受？
                                                            2. 名詞混淆：你對「呼召」與「職業」的理解，是否也受到了更正教扭曲的影響？這兩者有何本質區別？
                                                            3. 事奉的陷阱：為何「服事神」（全職事奉或教會服事）有時反而會潛藏與「主要呼召」的衝突？（提示：忙於做工卻忽略了與主的關係）。
                                                            4. 工作觀重塑：聖經如何看待工作？若恢復正確的呼召觀，將如何改變你的職涯規劃？
