                                                            尋找的被尋見 #

                                                            靠著己力要尋找人生的目的，是找不到的。

                                                            達文西的龐大失敗指出了一件事：只有追尋比人類需求更高的東西才得以滿足。「呼召」意味著一個驚人的反轉——追尋者（Seeker）本身才是被尋找的對象。

                                                            一、歷史的借鏡：達文西與佛羅倫斯 #

                                                            文藝復興時代藝術史學家 Vasari 曾這樣評論佛羅倫斯：

                                                            「佛羅倫斯對待藝術家，如同時間對待它的創造：創造，再緩緩地破壞、吞噬他們。」

                                                            這提醒我們，人類偉大的成就若缺乏更高的連結，終將面臨虛無。

                                                            二、什麼是「真正的追尋者」？ #

                                                            在西方語境中，「追尋者」常指靈性無歸依者。但作者定義的「真追尋者」具有以下特質：

                                                            1. 內在動力：能感受目的、活力、誠實、理想，和求答案的渴望。
                                                            2. 關鍵時刻：生命會來到一個徬徨、疑惑、困擾的節骨眼。
                                                            3. 必要的懷疑：因過去相信的答案無法回答新問題，而開始不信過去所信的（這是必經過程）。

                                                            三、關於「追尋」的四種論點 #

                                                            面對追尋，人類歷史上主要出現了四種態度：

                                                            1. 過程論：追尋就是一切，結果不重要 #

                                                            • 常見對象：受過教育、能變革者。
                                                            • 特徵：對終極答案存疑，高舉寬容、開放、模稜兩可。
                                                            • 盲點：
                                                              • 所謂「思想開放」往往變成了「沒有思想」。
                                                              • 所謂「寬容」往往變成了「不信任何事物」。
                                                            • 意象：如同傳說中「飄泊的荷蘭人」（Flying Dutchman），受詛咒永遠在海上漂流，無法靠岸。

                                                            2. 滅慾論：渴望本身就是問題（古代南亞觀點） #

                                                            • 論點：渴望是種根本邪惡，將我們束縛於痛苦和幻象中。
                                                            • 解法：不在於滿足渴望，而是制止它，最終達到涅槃。
                                                            • 盲點：對於肯定世界文化的人來說，這種徹底的避世主義難以接受。

                                                            3. Eros 之道（愛）：人類的「偉大昇華」 #

                                                            • 論點：追尋是人類朝自己渴想目標發展出的過程。
                                                            • 路徑：目標散發迷人特質 $\rightarrow$ 引發愛 $\rightarrow$ 轉化為渴望 $\rightarrow$ 轉化為擁有 $\rightarrow$ 轉化為快樂。

                                                            4. Agape 之道（愛）：神的「紆尊俯就」 #

                                                            • 論點：是愛找出了追尋者——無關其價值功過。
                                                            • 路徑：這是耶穌介紹的路，與 Eros 截然不同。

                                                            四、關鍵對比：Eros vs. Agape #

                                                            這是本章最重要的神學對比，區分了人本的宗教與神的呼召：

                                                            比較項目Eros 之道 (人找神)Agape 之道 (神找人)
                                                            方向由下而上的昇華由上而下的俯就
                                                            追尋的目標快樂可在尋找處尋見唯有神能永遠滿足我們
                                                            追尋的方法自己正尋找某樣事物自己是被人尋見

                                                            五、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 分辨：「真實的尋道者」和「靈性的遊蕩者」有何不同？
                                                            2. 懷疑：為何「不信」反而是「真實尋道者」的必經階段？
                                                            3. 檢視：為何許多尋道者很快就從前兩種看法（過程論、滅慾論）中轉離？它們缺乏什麼？
                                                            4. 體驗：你有過 C.S. Lewis 所描述的「喜樂驚魂」經歷，或「神來親近我」的感受嗎？那是怎樣的光景？你如何回應？
