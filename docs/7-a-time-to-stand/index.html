                                                            堅立的時刻 #

                                                            在現代世界裡，耶穌基督的信徒面對人類史上最強大、及世界上第一個真正的全球文化。

                                                            蒙田（Montaigne）評論溫泉關之役時說：「有種失敗可與勝利匹敵。」 在現代社會，我們需要的不是廉價的成功，而是如同錨一般穩住、如同食物一般滋養我們的真理，讓我們敢於在挑戰中冒險堅立。

                                                            一、信仰的三個關鍵時刻 #

                                                            作者將信仰的歷程區分為三個層次，指出真正的呼召在於最後一個階段：

                                                            1. 行為時刻 (The Moment of Behaving)

                                                              • 定義：照主教導去信。
                                                              • 特徵：這是基礎的教義認知與遵守。

                                                            2. 堅立時刻 (The Moment of Standing)

                                                              • 定義：照主期望去行。
                                                              • 特徵：這是在生活行為上的實踐與站立。

                                                            3. 相信時刻 (The Moment of Believing)

                                                              • 定義：從狹小舒適中走出，明白蒙召是最深刻、震撼且強烈的時刻。
                                                              • 特徵：這是冒險的時刻，也是回應呼召的頂峰。

                                                            二、當代教會的困境：為何影響力微乎其微？ #

                                                            這是一個針對現代信徒的靈魂拷問：

                                                            「若許多人自稱倚靠福音，但對社會的影響力卻很小，自身也如此乏味、微不足道：是福音的問題，還是我們的裡面有問題？」

                                                            問題的根源：信仰的私有化 #

                                                            為什麼會有這種現象？因為一種關於信心的錯誤觀點正大受歡迎：

                                                            • 錯誤觀點：認為信心只屬於「個人」、「關係」、「屬靈」及一些「簡單層次」。
                                                            • 後果：
                                                              • 全球經濟制度與世俗文化主導了公共領域。
                                                              • 信徒退縮到私領域，導致信仰與公共生活脫節。

                                                            三、解決之道：重尋呼召 #

                                                            面對現代社會的巨大推力，我們需要什麼？

                                                            • 呼召與福音：這是回應挑戰的最佳答案。
                                                            • 真理的功能：
                                                              • 像錨：在動盪中穩住你。
                                                              • 像食物：在匱乏中滋養你。
                                                              • 價值：讓你看重它超越一切，甚至激勵你去冒險。

                                                            四、溫泉關的啟示 #

                                                            蒙田對「溫泉關之役」（斯巴達三百壯士）的評論提醒我們：

                                                            • 失敗中的勝利：有一種失敗，因著其犧牲與堅守的價值，足以與勝利匹敵。
                                                            • 英雄的鼓舞：犧牲的英雄之所以能激勵我們，是因為他們展現了超越自我的呼召感。

                                                            五、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 重新定義成敗：蒙田這句「有種失敗可與勝利匹敵」是什麼意思？這觀點如何挑戰當代只求「贏」的文化？
                                                            2. 檢視環境：全球經濟制度如何潛移默化地影響了你的價值觀？作者認為現今教會面臨的最大挑戰是什麼？
                                                            3. 信仰檢測：為什麼將信心侷限在「個人、關係、屬靈」層次會如此受歡迎？作者為何反對這種觀點？
                                                            4. 文化變革：你認為信心能改變文化的潛在原因是？你能舉出歷史或現代的例子嗎？
                                                            5. 自我定位：你是否意識到自己正站在能改變世界的位置上？目前有什麼恐懼讓你退縮嗎？
