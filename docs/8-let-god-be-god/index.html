                                                            讓主作主 #

                                                            只有當我們敬畏神超過我們對世界的重視，我們才能讓信仰恢復權柄。

                                                            信仰要重獲主權，只有一個條件：當基督徒敬畏神超過懼怕現代力量及好處時。「讓主作主」（Let God be God）意味著面對生命中那個不容拒絕的決定性主權。

                                                            一、神的主要呼召：雙重層次 #

                                                            神所說的話（呼召），永遠包含兩個不可分割的層次，不能只取其一：

                                                            1. 既是 召喚 (Summons)，也是 邀請 (Invitation)。
                                                            2. 既是 法律 (Law)，也是 恩典 (Grace)。
                                                            3. 既是 要求 (Command)，也是 給予 (Gift)。

                                                            反思：我們常傾向只接受「邀請、恩典、給予」，卻忽略了「召喚、法律、要求」。

                                                            二、神聖的矛盾：可怕的甜美 #

                                                            作者提出了一個極具張力的觀點，解釋為何神的本質對現代人來說難以接受：

                                                            • 對不信者而言：猶太人對神本質的了解（絕對的主權、公義）似乎是「殘酷的」。
                                                            • 耶穌的本質：耶穌的甜美被稱為是「可怕的」（Terrible Sweetness）。因為那種愛是絕對的，不容許任何偶像並存。
                                                            • 西乃山的核心：「呼召」是西乃山立約事件的核心，神在此確立了祂對百姓的絕對主權。

                                                            三、恢復信仰的主權 #

                                                            面對現代世界的挑戰，我們該如何站立？

                                                            • 現代的挑戰：世界用各種力量（經濟、輿論、舒適）威脅我們，試圖動搖神的權威。
                                                            • 關鍵轉折：唯有當我們對神的敬畏大過對世界的恐懼時，信仰才能在我們的生命中重掌王權。
                                                            • 行動：認識生命中那個不容拒絕的決定性主權，並做出回應。

                                                            四、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 神學反思：為何猶太人對神本質的了解，對不信者來說顯得「殘酷」？（提示：因為它粉碎了人的自我中心）。
                                                            2. 耶穌的本質：為何耶穌的甜美是「可怕的」？這句話如何挑戰了當代「聖誕老人式」的上帝觀？
                                                            3. 自我檢測：在神呼召的兩個層次中（如法律 vs. 恩典），你較傾向回應哪個層次？只偏重一邊會產生什麼結果？
                                                            4. 權威之爭：現代世界在哪些方面挑戰了神的權威？你在生活的哪些領域感到這種張力（例如：職場倫理、金錢觀）？
                                                            5. 定義主權：「讓主作主」對你個人的具體意義是什麼？

                                                            📖 聖經默想：馬太福音 4:18-22 #

                                                            閱讀耶穌呼召彼得和安德烈的場景：

                                                            • 命令：耶穌向他們說了甚麼，以至於他們「立刻」捨去所有一切？
                                                            • 對照：這事件如何說明主要呼召（跟隨耶穌）和次要呼召（得人如得魚）的關係？
                                                            • 順服：門徒對耶穌權柄的順服，說明了呼召具有什麼樣的強制力與吸引力？
