                                                            將所有思想化為感恩 #

                                                            生命中沒有任何事物應該被視為理所當然，它必須以感恩的心領受。

                                                            現代世界最核心的特徵，就是幾乎完全沒有倚賴性和感恩的心，取而代之的是「強化遺忘」和「不知感恩」。 呼召要求我們像愛因斯坦一樣，深刻意識到自己欠了極大的債，因此感恩必須是我們對神的第一個回應，也是持續的回應。

                                                            一、愛因斯坦的信條：我知道我欠債 #

                                                            愛因斯坦在演說《我的信條》（My Credo）中，表達了一種深刻的謙卑：

                                                            「我時常因為想到我的生活大部分要倚賴與我同為人類者的工作，而感到不安，我知道我欠他們極大的債。」

                                                            這引出了一個根本的問題：生命中的回報是甚麼？我們是認為自己配得一切，還是充滿虧欠感？

                                                            二、現代世界的崩壞：從倚賴到權利 #

                                                            傳統世界雖然有罪，但並未拋棄對感恩極重要的兩件事；然而現代世界卻徹底破壞了它們：

                                                            比較項目傳統世界現代世界
                                                            對生命的認知感受到生命完全的倚賴性將倚賴感轉變成自治意識 (Autonomy)
                                                            對他人的認知感受到道德負債 (Moral Debt)將負債感轉變成權利和應得權利 (Rights & Entitlement)

                                                            現代的核心：這是一種**「強化的遺忘」**。現代人認為一切都是自己賺來的，因此不需要向任何人或神感恩。

                                                            三、呼召與感恩的連結 #

                                                            當呼召進入生命，它會在兩方面重塑我們與神的關係：

                                                            1. 恩典的流動：提醒我們，既有這麼大的恩典賞賜給我們，也應把恩典給其他人。
                                                            2. 持續的回應：感恩必須是我們對神的第一個回應，也是持續的回應。

                                                            四、榮譽勳章 vs. 深不可測的奇妙 #

                                                            這是對蒙召者最後的靈性拷問：

                                                            • 錯誤態度：你是否配戴你蒙揀選的身分，像「榮譽勳章」一樣？將它視為對你高尚生命的讚美？
                                                            • 正確態度：或者，呼召那深不可測的奇妙是否緊緊抓住你，像一股你絕對無法滿足或擺脫的偉大力量？
                                                            • 愛的激動：你是否曾經被呼召你的基督的愛所激動，超越你的自我？

                                                            五、小結：像那有罪的女人 #

                                                            耶穌是否能夠稱讚你，像祂稱讚那位用眼淚為祂洗腳的婦人一樣？ 她愛的多，因為她知道自己被赦免的多，被賜予的多。

                                                            聆聽拿撒勒人耶穌的聲音，回應祂的呼召。

                                                            六、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 感恩的體驗：你曾否被喚起要深深感謝某人或某事的經驗？你當時如何表達感激？
                                                            2. 理所當然：生命中的哪些事情（健康、陽光、空氣、救恩）被你認為是理所當然的？為什麼我們容易陷入這種「強化的遺忘」？
                                                            3. 社會觀察：為何對神的普遍排斥，使現代社會只「偶爾」感謝祂？這種不知感恩的文化如何表現在社會的權利運動或人際關係中？
                                                            4. 恩典的明證：在我們的生活中，呼召本身如何成為上帝恩典的明證？
                                                            5. 恩典的出口：為何「真正的感激」會自然地使我們想施恩於別人？
                                                            6. 具體表達：你能怎樣具體表達對神的感激？（不僅是言語，也包含行動）。

                                                            📖 聖經默想：路加福音 7:36-50 (有罪的女人膏主) #

                                                            • 愛的行動：你如何解釋這婦人獻上香膏、用眼淚洗耶穌腳的舉動？（她是為了換取赦免，還是因為已經被赦免？）
                                                            • 比喻含義：耶穌為何要講「欠債僕人」的比喻（欠五十兩與五兩）？
                                                            • 態度的對比：這裡如何描述西門（法利賽人）對待賓客的態度？他為何冷淡？
                                                            • 價值判斷：為何你會覺得這婦人在耶穌身上「浪費」香膏？這反映了我們對價值的什麼看法？
                                                            • 回應十架：這件事情說明我們應如何回應十字架的恩典？
