                                                            更快、更多、更好 #

                                                            資本主義在打敗所有挑戰之後，面對最大的挑戰是－－它自己，因為它吞吃了使它成功所需的一切美德。

                                                            十八世紀的清教徒領袖科頓·馬瑟（Cotton Mather）曾發出著名的警告：「呼召勢必帶來繁盛，但除非有警覺性，否則最後繁盛的結果會破壞呼召。」 這就像是一句古老的格言：「宗教生出富裕，女兒（富裕）卻毀了母親（宗教）。」

                                                            一、資本主義的內在危機 #

                                                            資本主義在打敗所有外部挑戰後，現在面對最大的挑戰就是它自己。

                                                            • 自我吞噬：因為它吞吃了使它成功所需的一切美德（如節制、勤奮、延遲享樂）。
                                                            • 清教徒的轉變：
                                                              • 早期清教徒：對世界無動於衷，結果容易獲得成功。
                                                              • 後來清教徒：進入世界做善事，而且做得「很好」（變得富有），重心卻逐漸偏移。

                                                            二、關於金錢的現代迷思與真相 #

                                                            要理解呼召，必須先除去關於金錢的兩項現代迷思：

                                                            1. 迷思一：認為「賺錢」比「金錢的意義」更加重要。
                                                            2. 迷思二：認為金錢是「中性」的，只是交易的媒介。

                                                            金錢的真相：

                                                            • 屬靈問題：金錢過去是屬靈問題，現在也是。瑪門（Mammon）是神的敵人。
                                                            • 替代性滿足：人們無法滿足地追求金錢，通常是因為將其視為其他需求的替代品：
                                                              • 權力 (Power)
                                                              • 保護 (Protection)
                                                              • 讚許 (Praise)
                                                            • 吞沒：追求金錢的個人和社會，很快會被金錢吞沒。

                                                            三、商品化的社會 #

                                                            當「不滿足」遇上「金錢」，就會加劇社會的商品化（Commodification）。

                                                            • 良善社會的指標：取決於有多少事物是超越市場價格的（即「非賣品」的層級和數量）。
                                                            • 愛的經濟化：「商品化」的社會最可怕之處，在於連愛都被經濟化了。
                                                            • 羅斯福的輕視：羅斯福總統曾說：「我努力地去喜愛卡內基，但是相當困難。除了將賺錢當作神的人，我不知道有哪種人更加令我感到輕視厭惡。」

                                                            四、呼召：對抗市場文化的防線 #

                                                            呼召對一個被金錢浸潤、市場主宰的文化，提出了兩個關鍵的制衡點：

                                                            1. 絕對主權：呼召意指在金錢和市場之上，有個絕對、即時的主權存在（神大過市場）。
                                                            2. 衝突邏輯：呼召在社會中引進的運作方式（恩典、給予），與市場心態（交易、獲利）直接衝突。

                                                            五、自我檢視：瑪門的權勢 #

                                                            耶穌指出，金錢會在我們生活中占有偶像的地位，直到它成為一個個人的、屬靈的、具像神一樣的力量，統治了我們。

                                                            • 你是否允許金錢主宰你的優先次序、評斷能力及人際關係？
                                                            • 你是否允許消費型的社會來定義你的需要？
                                                            • 關鍵行動：當你在做事情時，是否為神而做（為了對事情全然的愛）？是否能不受金錢影響，毫不猶豫地將金錢給予需要的人？

                                                            六、研讀與默想 (Action Plan) #

                                                            📌 研讀指引 #

                                                            1. 繁榮的腐蝕：呼召帶來繁榮，繁榮卻腐蝕呼召。你曾在歷史或現代教會中看到這種循環嗎？
                                                            2. 金錢屬靈論：為什麼作者堅持金錢是「屬靈的問題」而非中性的工具？你曾有過什麼經歷可以證明這個說法？
                                                            3. 貪婪的寬容：現代社會已被金錢宰制，我們對「貪婪」是否變得過於寬鬆？
                                                            4. 非賣品：為什麼「出售」和「不出售」之間的界線，是衡量社會價值的指標？在你心中，有哪些東西是絕對不出售的？
                                                            5. 愛的替代品：人類如何嘗試用財產和必需品來取代愛和關係？

                                                            📖 聖經默想：馬可福音 10:17-31 (少年財主) #

                                                            • 核心缺乏：耶穌對那位富有的少年官說「你還缺少一件」，那一件是什麼？
                                                            • 變賣所有：為什麼耶穌要求他變賣「所有的」？這是對所有富人的普遍要求，還是針對他個人的偶像（瑪門）所發出的特定呼召？
                                                            • 進神國：為何富人進神的國如此困難？這與「倚靠金錢」所帶來的假性安全感有何關係？
